# Butter Bear — JetBrains

Works in IntelliJ IDEA, PyCharm, WebStorm, GoLand, CLion, Rider, and all other JetBrains IDEs.

## Install

1. Open your JetBrains IDE
2. Go to **Settings → Editor → Color Scheme**
3. Click the gear icon → **Import Scheme**
4. Select the `.icls` file for your preferred variant
5. Apply

## Variants
- `butter-dark.icls` — Deep warm amber (flagship)
- `butter-light.icls` — Warm cream for bright environments
- `butter-soft.icls` — Desaturated for 2am sessions
- `butter-midnight.icls` — Navy + amber
- `butter-mocha.icls` — Espresso brown
- `butter-storm.icls` — Steel blue + amber (Tokyo Night style)
- `butter-rose.icls` — Warm rose + amber
- `butter-forest.icls` — Deep forest green + amber
- `butter-amoled.icls` — Pure black for OLED displays
- `butter-hc.icls` — Maximum contrast (WCAG AAA)
